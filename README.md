## How to run the algorithmn?
1. run alg.py
2. install Node.js
3. run `npm install -g yarn` to install yarn
4. run `yarn` or `yarn --dev` to add all dependences
5. use `yarn start` to run server.js to set API
6. go to localhost:3000 to view the API
